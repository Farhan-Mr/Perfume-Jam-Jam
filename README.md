# Perfume e-Farhat

A modern, responsive HTML + CSS website for a premium perfume e-commerce store, built with Bootstrap 5 framework.

## Project Overview

This project demonstrates a professional, full-featured e-commerce website template for a perfume store. It includes responsive design, modern styling, and interactive features.

## Features

- **Responsive Design**: Mobile-first approach with Bootstrap 5
- **Modern UI**: Clean and professional interface
- **Navigation**: Smooth scrolling navigation
- **Product Display**: Featured products showcase
- **Contact Form**: Customer contact section
- **Bootstrap Framework**: Leverage Bootstrap 5 components
- **Custom CSS**: Tailored styling and animations
- **JavaScript**: Interactive functionality

## How to Get Started

1. **Open the project** in VS Code
2. **View the website** using Live Server extension (recommended)
   - Install "Live Server" extension
   - Right-click `index.html` and select "Open with Live Server"
3. **Customize** the HTML, CSS, and JavaScript files as needed

## Customization Guide

### Colors
Edit the CSS custom properties in `css/style.css`:
```css
:root {
    --primary-color: #7c3aed;
    --secondary-color: #ec4899;
    --dark-color: #1f2937;
    --light-color: #f3f4f6;
    --text-color: #374151;
}
```

### Content
Update text, images, and links directly in `index.html`

### Functionality
Extend JavaScript features in `js/script.js`
